minisamについて

Pairs are separated by ~300 characters (more or less).

Start and end position define the original string from the chromatid (before simulation). Then there are some simulated substitutions/deletions/insertions which return the read string in the minisam file. Since there are deletion and insertion, (end-start) is not always 150 char, but the generated read will always be 150 char.

Minisam files just provides the reads true positions. It is up to you to define your confidence scores.
